# Contributing to SolarFarmer API Python SDK

Thank you for considering contributing to our project! Please follow these steps:

1. Fork the repository and create a new branch
2. Ensure your code adheres to our coding standards
3. Write clear commit messages and include tests for new features
4. Submit a pull request with a detailed description of your changes

You can open an issue or email solarfarmer@dnv.com to check if we are already working on your feature of interest.
